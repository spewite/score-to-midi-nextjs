# 🎵 Score-to-MIDI (Frontend)

This is the **frontend** of the Score-to-MIDI project, a web application that allows users to upload sheet music images and convert them into MIDI files.

## 📦 Installation & Setup

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (Developed in: `v20.17.0`)

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/spewite/score-to-midi-nextjs
cd score-to-midi-nextjs
npm install --force
```

### Configuration

Create a .env file and set the backend API URL:

```bash
NEXT_PUBLIC_BACKEND_URL=http://127.0.0.1:5000
```

### Run the project 

```bash
npm run dev
```
